# Practical ML Lab

**Hands-On Machine Learning** (Géron) — feature engineering, model selection, hyperparameter tuning, ensembles, cross-validation, production ML.

Uses **ml_toolbox.textbook_concepts.practical_ml** (FeatureEngineering, ModelSelection, HyperparameterTuning, EnsembleMethods, CrossValidation, ProductionML).

## Run (from repo root)

```bash
python learning_apps/practical_ml_lab/app.py
```

Open **http://127.0.0.1:5008**

## Contents

- **Feature Engineering**: polynomial features, interactions, binning
- **Model Selection**: select classifier/regressor by criteria
- **Hyperparameter Tuning**: grid/random search
- **Ensemble Methods**: voting, stacking
- **Cross-Validation**: stratified K-fold, time-series splits
- **Production ML**: deployment patterns

Learn by book or level. Demos can be added in demos.py.
