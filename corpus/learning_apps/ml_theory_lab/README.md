# ML Theory Lab

**Understanding Machine Learning** (Shalev-Shwartz & Ben-David) — PAC learning, VC dimension, generalization bounds, stability, Rademacher complexity.

Theory-focused curriculum; optional code/demos can be added in demos.py.

## Run (from repo root)

```bash
python learning_apps/ml_theory_lab/app.py
```

Open **http://127.0.0.1:5011**

## Contents

- **PAC Learning**: (ε, δ)-PAC, agnostic PAC, sample complexity
- **VC Dimension & Generalization**: shattering, VC bounds
- **Stability & Rademacher**: uniform stability, Rademacher complexity
- **Generalization Bounds**: bias-complexity tradeoff

Learn by book or level.
