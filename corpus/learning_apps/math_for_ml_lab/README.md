# Math for ML Lab

**Basic math for machine learning**: linear algebra, calculus, probability & statistics, optimization.

Uses **ml_toolbox.math_foundations** (linear_algebra, calculus, probability_statistics, optimization).

## Run (from repo root)

```bash
python learning_apps/math_for_ml_lab/app.py
```

Open **http://127.0.0.1:5013**

## Contents

- **Linear Algebra**: vectors (dot, norm), matrices (multiply, transpose, inverse, det, rank), SVD, eigendecomposition
- **Calculus**: numerical derivative, gradient, Jacobian, Hessian
- **Probability & Statistics**: Gaussian, Bayes' rule, MLE, expectation, variance, covariance
- **Optimization**: gradient descent, SGD, Adam, convex optimization

Learn by book or level. Demos: dot product, SVD, derivative, gradient, Gaussian sample, gradient descent.
