## Troubleshooting

### **"Python not found"**
- Install Python from https://www.python.org/
- Make sure to check "Add Python to PATH" during installation

### **App won't start**
- Try the Simple Desktop version first (START_SIMPLE_APP.bat)
- It doesn't need any extra packages

### **Web app port error**
- Close other apps using port 5000
- Or change port in ml_learning_app.py (line with `port=5000`)
